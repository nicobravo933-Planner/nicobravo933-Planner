18. ¿Aportamos Valor? (Forecast Value Added)

Felicidades. Has construido modelos de Machine Learning, has simulado redes complejas y has optimizado inventarios bajo restricciones. Pero ahora te enfrentas al componente más difícil de gestionar en la Supply Chain: el ser humano.

En la mayoría de las empresas, el número que sale del algoritmo no es el que se usa para comprar. Ese número viaja por salas de juntas, es ajustado por gerentes comerciales “que conocen al cliente” y modificado por directores de finanzas que necesitan “llegar al presupuesto”. En este capítulo, aprenderemos a medir si todos esos toques humanos realmente aportan valor o si son simplemente ruido costoso mediante el Forecast Value Added (FVA).

18.1 La Auditoría del Proceso de Planificación

El Forecast Value Added (FVA) no es una métrica de error; es una métrica de proceso. Su objetivo es identificar qué pasos del proceso de planificación mejoran la precisión y cuáles la empeoran.

En un proceso típico de S&OP (Sales and Operations Planning), el número evoluciona así:

  1. Pronóstico Estadístico (Baseline): Lo que dice la matemática (ARIMA, XGBoost).

  2. Ajuste del Demand Planner: El técnico que limpia promociones o eventos conocidos.

  3. Ajuste Comercial/Ventas: “Siento que este mes venderemos más por el mundial”.

  4. Consenso Final: El número que se firma en la reunión directiva.

El FVA calcula la diferencia de error entre cada uno de estos pasos. Si el ajuste comercial aumenta el error en un 5% respecto al modelo estadístico, ese ajuste ha destruido valor.

18.2 El Juez Supremo: El Pronóstico Ingenuo (Naive)

Para que el FVA tenga sentido, necesitamos un punto de comparación absoluto. Michael Gilliland, el padre del FVA, sugiere usar el Pronóstico Ingenuo (lo que vendí el mes pasado es lo que venderé este mes) como el primer eslabón.

Si tu sofisticado modelo de Deep Learning no es capaz de ganarle a un promedio móvil simple o al dato del mes pasado, entonces tu arquitectura tecnológica está destruyendo valor. El FVA nos obliga a ser humildes: si no puedes mejorar la simplicidad, no compliques el sistema.

18.3 El Costo del “Toqueteo” (Overfitting Humano)

¿Por qué los humanos suelen destruir valor en el pronóstico?

  • Sesgo de Optimismo: Los vendedores tienden a sobreestimar las ventas para asegurar que haya inventario “por si acaso”.

  • Presión Presupuestaria: Finanzas ajusta el pronóstico hacia arriba para alinearlo con los objetivos de crecimiento, confundiendo un objetivo con una predicción.

  • Efecto Reciente: Un planificador ajusta todo el mes porque ayer un cliente canceló un pedido, ignorando la tendencia a largo plazo.

El FVA expone estos comportamientos. Al final del trimestre, el reporte de FVA muestra: “Ventas ajustó 200 productos; en 180 de ellos el error aumentó”. Esa es una conversación poderosa que cambia la cultura organizacional de una basada en opiniones a una basada en evidencia.

18.4 Simulador: Analizador de FVA

En este laboratorio, comparamos tres etapas: la Matemática, el Ajuste Humano y la Realidad. Ajusta qué tan “optimista” o “pesimista” es el humano y observa si está añadiendo valor o destruyendo la precisión del sistema.

Plotly = require("plotly.js-dist-min@2.24.1")

viewof stat_forecast = Inputs.range([50, 200], {value: 110, step: 5, label: "Modelo Estadístico:"})
viewof human_adjustment = Inputs.range([-50, 50], {value: 30, step: 5, label: "Ajuste Humano (±):"})
final_forecast = stat_forecast + human_adjustment
err_stat = Math.abs(actual_demand - stat_forecast)
err_final = Math.abs(actual_demand - final_forecast)

// FVA: Si el error final es menor al estadístico, el valor es positivo
fva_value = err_stat - err_final 
fva_pct = (fva_value / actual_demand) * 100

18.5 Implementando el FVA en tu Organización

Para que el FVA funcione como herramienta de cambio, debe seguir estas reglas:

  1. No es para castigar: El objetivo es aprender dónde el juicio humano es valioso (ej. lanzamientos, promociones) y dónde es perjudicial (ej. productos estables).

  2. Métrica de Ganancia: Mide el FVA en dinero. “El ajuste de Ventas nos costó $50,000 extra en inventario este mes”. Esa cifra se entiende en cualquier nivel directivo.

  3. Eliminación de Pasos: Si un paso del proceso (ej. la revisión regional) destruye valor consistentemente durante 6 meses, ese paso debe ser eliminado del flujo de trabajo.

El FVA es el filtro que permite que la Arquitectura de Suministros se mantenga limpia, eficiente y libre de sesgos políticos.
